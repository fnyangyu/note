react 事件处理

  1. 给 react 组件中的节点绑定事件，必须使用行内事件方式绑定，事件名称 驼峰方式
  2. 有个问题，就是挡在行内绑定事件的时候，事件函数内部的 this 就会指向 window， 所有在组件内部 生命事件函数的时候，通常使用 箭头函数
  3. 也可以使用 constructor 方法在其内部改变事件函数的this指向
  4. react 的事件函数的最后一个参数默认是 事件对象event

代码

    return (
          <div>
            {/*
              非箭头 函数 传参
              <button onClick={this.handleClick.bind(this,a)}>按钮</button>
              */}
            <button onClick = { () => this.handleClick(a) }>
                按钮
            </button>
            <input onChange = { this.handleClick }></input>
          </div>
        )

组件调用（父组件给子组件传）

  1. 组件默认有一个 props 属性，这个属性值是一个对象
  2. 该对象内部存储的是父组件 传过来的值
  3. 在子组件内部获取 props 使用 （类里边调用自己的属性用this）
  4. props 是只读的，不可更改
  5. props 对象下有一个 children 默认属性

在组件内部插入另一个组件：

  App.js中：
    <wrap>
      <Button />
    </wrap>

  Wrap.js中：
    <div>
      { this.props.children }
    </div>
