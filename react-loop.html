react路由

react-loop 路由

    import {BrowserRouter as Router, Route, Link} from 'react-router-dom'//引入路由
    <Router>
        <div>
            <Route path='/' component={Home} />
            <Route path='/about' exact component={About} />{//exact 是严格匹配路径}
        </div>
    </Router>
  • 路径名全部不能大写;{${path}/:name} ------':name' 是一个变量 在路径里边变量就这么写
  • 路由 Route父组件默认会传location history match 这三个props
  • Switch 的作用是单一匹配 ，只要匹配到一个就不再往下匹配。
  • Redirect 重定向
  • history 中有三个常用的方法：history.goBack() history.push(路径)（跳到路径页）

react-ajax

  • ajax 就是用js实现页面的局部刷新 (fetch) 原生的升级版 $.ajax axios xml 常用的获取数据

  • JSON:

    1. JavaScript 对象表示法（JavaScript Object Notation）
    2. JSON 是存储和交换文本信息的语法。类似 XML
    3. JSON 比 XML 更小 更快，更易解析。
    4. 将json格式转化为对象类型 JSON.stringify()（将对象转化为json字符串） JSON.parse(objJson)(将json字符串转化为对象)
    5. post 方法向服务器发送数据。设置发送数据的类型：contentType:'application/json'

  • Promise

    1. pending 执行中
    2. resolve 执行完毕
    3. reject 执行失败
