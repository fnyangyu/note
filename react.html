react

react 笔记

１．组件

  • 函数式声明组件，函数名首字母必须大写

        function() Welcome(){
      return <h1>hello</h1>
    }
    ReactDom.render(<Welcome/>,document.getElementById('root'))

  • 组件不一定复用，组件代码100行左右

        import App from './App.js'
    let Welcome = () => <h1>hello</h1>
    ReactDom.render(App,document.getElementById('root'))

  • props

    • react 组件内部有一个默认的参数叫props - 作用是接收父组件传过来的参数
    • 如果是函数组件的话，函数的第一个参数默认就是props
    • 如果是es6 class组件的话，组件内部使用 this.props 获取props
    • props 对象是只读的 不能修改
  • 当父组件传给子组件一个 react 节点的时候，子组件使用 this.props.children 接收

2.import React from 'react' 引入是为了支持jsx语法 import ReactDom from 'reactDom' 方便
